# Python Mini Projects

Dieses Repository enthält zwei einfache Python-Konsolenprogramme,
die im Rahmen meines Selbststudiums erstellt wurden.

## 1. Alter-65-Rechner

### Beschreibung
Dieses Programm berechnet das aktuelle Alter einer Person
und zeigt, in welchem Jahr die Person 65 Jahre alt wird.

### Funktionen
- Abfrage des Geburtsjahres
- Berechnung des aktuellen Alters
- Berechnung des Jahres, in dem man 65 Jahre alt wird
- Ausgabe des Ergebnisses in der Konsole

### Verwendung
```bash
python Alter_65.py

## 2. Taschenrechner
