# Alter-65-Rechner

## Beschreibung
Dieses Projekt ist ein einfaches Python-Konsolenprogramm.
Es berechnet das Alter einer Person und zeigt,
in welchem Jahr die Person 65 Jahre alt wird.

## Funktion
Das Programm:
- fragt nach dem Geburtsjahr
- berechnet das aktuelle Alter
- berechnet das Jahr, in dem man 65 Jahre alt wird
- gibt das Ergebnis in der Konsole aus

## Verwendung
1. Terminal im Projektordner öffnen
2. Programm starten mit:
   python Alter_65.py
3. Geburtsjahr eingeben
4. Ergebnis lesen

## Beispiel
Eingabe:
1999

Ausgabe:
Du bist ungefähr 26 Jahre alt.
Du wirst 65 Jahre alt im Jahr 2064.

## Lernziele
- Variablen
- input() und print()
- Rechnen mit Zahlen
- Arbeiten mit datetime
- Python über die Konsole ausführen
